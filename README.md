# 🧮 Calculadora de IMC em Kotlin

Aplicação simples desenvolvida em Kotlin que calcula o Índice de Massa Corporal (IMC) a partir do peso e altura informados pelo usuário.

## 📌 Sobre o projeto

Este projeto foi criado com o objetivo de praticar lógica de programação e entrada de dados no console, utilizando a linguagem Kotlin. A aplicação permite que o usuário insira seus dados e receba como retorno o valor do IMC junto com sua classificação.

## 🚀 Funcionalidades

* Cálculo automático do IMC
* Entrada de dados pelo usuário (peso e altura)
* Exibição do resultado com classificação:

  * Abaixo do peso
  * Peso normal
  * Sobrepeso
  * Obesidade

## 💻 Tecnologias utilizadas

* Kotlin

## ▶️ Como executar

1. Abra o projeto na IDE (IntelliJ)
2. Execute o arquivo principal (`main.kt`)
3. Insira os dados solicitados no console

## 📚 Fórmula utilizada

IMC = peso / (altura × altura)

## 🎯 Objetivo

Praticar conceitos básicos de programação, como:

* Variáveis
* Condicionais
* Entrada e saída de dados
